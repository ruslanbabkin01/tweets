## Description
The application has 2 pages Home and Tweets. The Tweets page loads user cards from the database. By clicking on the Follow button in the card, you can subscribe to a user or unsubscribe.

## Technologies
- **react**
- react-router-dom
- styled-emotions
- axios
- local-storage
